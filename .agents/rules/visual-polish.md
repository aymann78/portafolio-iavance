## Visual quality bar
This portfolio must feel premium, cinematic, technical, and commercially sharp.
Do not ship flat sections that are only cards on dark backgrounds.
Every important section should include at least one of:
- depth
- motion
- layering
- visual rhythm
- meaningful contrast
- interactive feedback

## Hero and large text rules
- Large headings must fit on a 1366x768 laptop without hiding the core message or CTA.
- Avoid oversized type that looks impressive only on very large monitors.
- Large headings may use gray-to-white progressive reveal on scroll, but never arbitrary mixed colors.
- If a heading uses muted text, the muted part must have a clear interaction or narrative reason.

## Animation rules
- Avoid fake motion that does not describe a real system behavior.
- Prefer motion that explains flow, state, transition, or hierarchy.
- Use restrained motion with clear intent.
- If an animation feels decorative but not informative, replace it.

## Flatness rule
If a page feels visually flat, improve it using a system of depth:
- subtle grid
- noise texture
- radial light
- layered gradients
- sticky transitions
- section separators
Do not add random effects with no systemic logic.
